## 42-FdF

Wire frame model project (fdf -> "Fil de Fer" in French) coded using minilibx library. 
You need to install the library.

Type 'make linux' if you use linux, otherwhise(for MacOS) use just make.


## run

	./fdf <map>

## zoom
	MOVE : wasd
	ZOOM : +/-
	HEIGHT : f/g keys
